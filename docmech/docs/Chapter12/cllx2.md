# 12.2 杆件拉压与材料的力学性能

## 拉压时的内力分布

### 截面法

&emsp;&emsp;对于一根受轴力的杆件，我们可以分别对其不同受力特性的段进行想象中的截断，丢弃两部分之一，则保留下来的一部分上的合力必然与这个想象中截面上的内力大小相同、方向相反。

&emsp;&emsp;比如下面这个简单的案例（杆身全长是同一个受力特性），用m-m截面截断杆身后，不管是舍去右边段得到与左边段外力平衡的内力，还是舍去左边段，得到的结果都是相同的，内力FN与外力F相平衡。。
